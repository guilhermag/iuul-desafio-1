Docker

Site oficial

Docker usa virtualização para entregar software em pacotes chamados containers.

Vantagens

  • Deploy rápido

    Diminui o tempo de deploy da aplicação.

  • Segurança

    Containers são completamente isolados.

  • Eficiente em integração contínua

    Funciona muito bem com ferramentas de CI/CD como Jenkins.

Desvantagens

  • Ainda faltam algumas funcionalidades

    Algumas funcionalidades ainda são procuradas.

  • Dados do contêiner

    Backup de dados do container pode ser difícil.

  • Questões relativas a segurança

    Questões como monitoramento da segurança do container pode ser uma tarefa árdua.

HTML

Site oficial

HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

Vantagens

  • Muito utilizado

    Amplamente utilizado para construção de páginas web.

  • Fácil de aprender

    Html é fácil de começar a aprender.

  • Rápido carregamentos

    Arquivos .html são bem leves.

Desvantagens

  • Estático

    Não produz conteúdo dinâmico sozinho.

  • Segurança

    Ferramentas de segurança no html são limitadas.

  • Complexidade

    Páginas podem ficar bem complexas e difíceis de entender.

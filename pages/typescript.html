Typescript

Site oficial

Um superset da linguagem javascript, que adiciona tipagens e outras características.

Vantagens

  • Previsibilidade de bugs

    Resolução de bugs antes do código ir para produção.

  • Suporte das IDES

    Várias IDEs suportam o typescript.

  • Programação orientada a objetos

    Suporta conceitos de POO.

Desvantagens

  • Não é 100% tipado

    Não é totalmente como Java e C# por exemplo.

  • Mais uma tecnologia para ser estudada

    Nova documentação e detalhes para serem estudados.

  • Mais um passo para o desenvolvimento

    O arquivo .ts precisa ser transpilado para .js.

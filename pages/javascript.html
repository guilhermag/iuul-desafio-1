JavaScript é uma linguagem de programação interpretada estruturada.

Vantagens

  • Rápido

    Por ser uma linguagem interpretada, dai sua rapidez.

  • Simplicidade

    Em sua maioria é fácil de aprender e entender.

  • Popularidade

    Javascript se tornou uma das linguagens mais populares.

Desvantagens

  • Segurança

    Códigos no lado do cliente podem ser perigosos.

  • Debugger

    Debugar um código na web pode ser complicado

  • Natureza permissiva

    Muita liberdade pode levar a má práticas e problemas.
